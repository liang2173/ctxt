# 🧠 AI 智记 — 沉浸式记忆刷题系统

基于大语言模型的智能记忆刷题工具。上传知识点文档，AI 自动生成多层级题目，结合间隔重复算法实现科学记忆。

## ✨ 核心功能

- **🤖 AI 智能出题** — 上传 Word/PDF/TXT 或粘贴文本，DeepSeek 自动拆解知识点，生成选择题、填空题、简答题三级题目
- **🧠 间隔重复算法** — 基于 SM-2 改进的调度引擎，遗忘临界点自动推送复习
- **📈 多层级递进** — 选择题→填空题→简答题→掌握，四级递进路径
- **🔪 斩杀与复活** — 已掌握知识点可斩杀隐藏，48 小时后自动复活巩固
- **📁 分项目管理** — 多学科独立题库，互不干扰
- **🧩 分批刷题** — 每 10 个知识点为一批，逐批解锁，避免无休止刷题
- **⭐ 了解型 & 错题本** — 标记不重要知识点仅做选择，错题集中攻克，独立刷题模式
- **📡 AI 出题全程可见** — 流式输出、进度条、阶段指示器，生成结果卡片预览
- **🔍 预检面板** — 生成前免费预检：识别条目数、预估 Token 消耗、图片检测、确认后再调用 API
- **📥 自动备份** — 每次导入自动下载备份文件，浏览器清缓存也不怕

## 🚀 快速开始

1. 下载 `app.html`
2. 用浏览器打开（Chrome/Edge 推荐）
3. 进入「设置」页，填入 [DeepSeek API Key](https://platform.deepseek.com/)
4. 切换到「题库」页，上传知识点文档或粘贴文本
5. 点「AI 生成题目」，等待生成完成
6. 切换到「刷题」页开始学习

## 📖 使用说明

### 导入知识点

支持三种方式：

- **上传文件**：拖拽或点击上传 `.docx` / `.txt` 文件（PDF 仅支持文字型，图片型 PDF 无法解析）
- **粘贴文本**：支持序号格式（`1、` `2.` `一、` 等），自动拆分知识点
- **手动导入**：适用已有格式题目，支持 `题目|答案` 或选择题格式

### 刷题流程

1. 选择题：建立基础识别，连对 2 次晋级
2. 填空题：强化关键术语记忆
3. 简答题：训练完整复述能力（可自评申诉）
4. 掌握后「斩杀」，48 小时自动复活巩固

### 项目管理

- 顶栏下拉切换项目
- 题库页新建、重命名、删除项目
- 每个项目独立统计和数据

### 数据安全

- 所有数据存储在浏览器本地
- 每次 AI 导入自动下载 `.json` 备份
- 设置页可随时手动导出/导入全部项目

## 🛠 技术栈

- 纯前端：HTML5 + CSS3 + Vanilla JavaScript
- AI 引擎：DeepSeek Chat API（流式输出）
- 存储：LocalStorage（支持自动迁移旧格式）
- 文档解析：自研 ZIP 流解析器（`.docx`）+ pdf.js（`.pdf`）

## 📋 知识点格式建议

推荐为每条知识点添加序号，AI 识别效果最佳：

```
1、拉采尔是近代人文地理学的重要奠基人之一，其学术贡献...
2、施吕特尔是景观学派的创始人，他认为...
3、赫特纳是区域地理学派的代表人物，主张...
```

支持的序号格式：`1、` `2.` `3）` `一、` `二、` 等。子条目（`a.` `(1)` `①`）会自动归入上级。

## ⚠️ 注意事项

- 需要有效的 DeepSeek API Key
- 单次上传建议不超过 65 条知识点（系统自动分片处理）
- PDF 仅支持文字型，PPT 导出的图片型 PDF 无法提取文字
- 清除浏览器缓存会导致数据丢失，请定期导出备份
